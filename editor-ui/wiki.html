Condition Wiki

Quick meanings and examples for supported condition types.

TypeMeaningExample
minDimension score must be at least value.{ "type": "min", "dim": "focus", "value": 5 }
max_leDimension score must be less than or equal to value.{ "type": "max_le", "dim": "wonder", "value": 3 }
max_geDimension score must be greater than or equal to value.{ "type": "max_ge", "dim": "calm", "value": 0 }
diff_greaterA must be greater than B plus value.{ "type": "diff_greater", "a": "calm", "b": "tempo", "value": 2 }
diff_abs_lteAbsolute difference between A and B must be less than or equal to value.{ "type": "diff_abs_lte", "a": "focus", "b": "insight", "value": 4 }
top_isSelected dimension must be the highest score.{ "type": "top_is", "dim": "charm" }
not_top_isSelected dimension must not be the highest score.{ "type": "not_top_is", "dim": "tempo" }
rank_isSelected dimension must be at exact rank (1 = highest).{ "type": "rank_is", "dim": "calm", "rank": 2 }
top_diff_gteTop score minus second score must be at least value.{ "type": "top_diff_gte", "value": 3 }
top_diff_lteTop score minus second score must be at most value.{ "type": "top_diff_lte", "value": 2 }
total_minSum of all dimension scores must be at least value.{ "type": "total_min", "value": 10 }
total_maxSum of all dimension scores must be at most value.{ "type": "total_max", "value": 20 }
sum_minSum of selected dimensions must be at least value.{ "type": "sum_min", "dims": ["calm", "focus"], "value": 6 }
sum_maxSum of selected dimensions must be at most value.{ "type": "sum_max", "dims": ["tempo", "wonder"], "value": 8 }
spread_betweenScore spread (max-min) must be within [min, max].{ "type": "spread_between", "min": 0, "max": 6 }

Tip: Unknown custom types are preserved in JSON but are not interpreted by the quiz runtime unless implemented there.
